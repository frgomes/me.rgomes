This package provides implementations of Preferences and PreferencesFactory interfaces.

This implementation does the following steps:

  • Retrieves MANIFEST.MF using ResourceLookup.
  • Obtains attribute realm from MANIFEST.MF or assumes "default" if it is not found.
  • Obtains attribute mode from MANIFEST.MF or assumes "debug" if it is not found.
  • Loads resource ./config/${realm}/${mode}/application.properties, if it exists.
  • Loads resource ./config/${realm}/${mode}/password.properties, if it exists.
Properties defined in password.properties supersedes properties defined in application.properties.
